## XIVLauncher & Dalamud
<div class="custom-tags">
    <div class="custom-tag unique">Unique</div>
</div>
XIVLauncher is a third-party launcher for FFXIV, and it comes with the ability to run the game with Dalamud which allows us to install third-party plugins.  
If you're unfamiliar with XIVLauncher, please note that talking about it in any official channels such as in-game or when using official hashtags on social media is VERY discouraged and could get you in trouble.  
The first rule of using Dalamud is to not speak about Dalamud.  
[Get XIVLauncher Here](https://goatcorp.github.io/) and read the entire first page.

## Textools
<div class="custom-tags">
    <div class="custom-tag character">Characters</div>
</div>
Textools is an older solution to exporting things from the game, but still works. It lets you export character's with PBR textures which can look smoother than the what we get out of Meddle, though we can achieve the same result without Textools by baking our textures. Worth knowing that it exists, but using the Meddle workflow is recommended.    
[Textools Website](https://www.ffxiv-textools.net/)
