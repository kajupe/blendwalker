# Finding Good Tutorials
{: .fs-9 .no_toc }
There's a lot out there and a lot of them suck.
{: .fs-6 .fw-300 }

Don't just go to the donut, that guy is kinda cringe.

Try to find ones that don't just go "do x then y" but actually explain why.

There are usually multiple ways to do the same thing. Skim multiple tutotials to see if they use different workflows.

Instead of just watching tutorials, start a project. Then when you get stuck, look up how to get yourself un-stuck.
